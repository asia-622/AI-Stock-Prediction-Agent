# 🚀 AI Stock Prediction Agent

**Premium LSTM-powered stock prediction system with unlimited file upload (200MB+), AI trading insights, and modern glassmorphism dashboard.**

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://share.streamlit.io/)

## ✨ Features

- **✅ Unlimited CSV Upload** - Handles 200MB+ files (1K to 1M+ rows)
- **🧠 LSTM Neural Network** - Production-ready RNN for time-series prediction
- **🤖 AI Trading Agent** - BUY/SELL/HOLD recommendations with risk analysis
- **📊 Premium Dashboard** - Dark glassmorphism UI with interactive charts
- **⚡ Memory Efficient** - Optimized for large datasets
- **🎯 Production Ready** - Error-free, deployable to Streamlit Cloud
